# Ibrahim Electric Album Covers

This directory contains album cover art for the music player.

## Adding Album Covers

To add actual album covers, download high-quality images for each album and place them in this directory with the following filenames:

### Required Album Covers

1. **rumours-from-outer-space.jpg** - Rumours From Outer Space (2014)
2. **brothers-of-utopia.jpg** - Brothers of Utopia (2008)
3. **absinthe.jpg** - Absinthe (2006)
4. **royal-air-maroc.jpg** - Royal Air Maroc (2010)
5. **isle-of-men.jpg** - Isle of Men (2012)
6. **the-marathon-concert.jpg** - The Marathon Concert (Live) (2016)
7. **meets-ray-anderson-again.jpg** - Ibrahim Electric Meets Ray Anderson - Again (2007)

## Where to Find Album Covers

You can find album cover art from:

- **Spotify**: Right-click on the album → "Show Album" → Use browser inspector to get the high-res image URL
- **Apple Music**: Browse the album and screenshot or use browser inspector
- **Discogs**: https://www.discogs.com/artist/779115-Ibrahim-Electric
- **MusicBrainz**: https://musicbrainz.org/artist/806a7828-5050-4ba0-a2ea-f5897038e6d9
- **Band's Official Website**: http://www.ibrahimelectric.com/

## Image Specifications

- Format: JPG or PNG
- Recommended size: 500x500 pixels minimum (higher is better)
- Aspect ratio: Square (1:1)
- File size: Keep under 500KB for faster loading

## Note

The player will display a default vinyl appearance if album covers are not present. The vinyl will still rotate and play music normally.
